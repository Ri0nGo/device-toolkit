# IOT Studio UI 功能说明汇总

## 软件定位

IOT Studio 是用于 ESP32-C3 网关/节点的桌面配置工具原型，主要通过 USB 串口完成设备扫描、硬件信息读取、配置预览和配置写入。

## 左侧区域

### 品牌区

- 显示软件名称：`IOT Studio`
- 左侧显示产品图标。
- 右侧显示三个点按钮。
- 点击三个点按钮打开“关于软件”弹框。

### 扫描串口

- 按钮文字：`扫描串口`
- 功能：扫描当前电脑可用串口，并识别运行目标固件的 ESP32-C3 设备。
- 扫描后在左侧显示设备列表。

### 设备列表

设备卡片显示内容：

- 串口号，例如 `COM5`
- 配置状态：`已配置` / `待配置`
- 芯片与 USB 类型，例如 `ESP32-C3 · USB CDC`
- MAC 地址
- 设备 ID，未设置时显示 `-`

点击设备后：

- 选中当前设备。
- 高亮设备卡片。
- 启用配置表单。
- 加载设备当前配置。
- 刷新硬件信息。

### 底部状态

- 显示已发现设备数量：`已发现: 0`
- 扫描后更新为实际设备数量。

## 中间配置区域

### 配置卡片

标题：`配置`

说明：`所有变更会先预览，再写入 NVS。`

右上角按钮：`读取当前`

功能：重新读取当前选中设备的配置并回填表单。

### 未选择设备状态

未选择设备时：

- 所有输入框禁用。
- 所有下拉框禁用。
- `恢复` 按钮禁用。
- `预览并确认` 按钮禁用。
- 设备角色默认显示第一个选项，但不可操作。

### Tab 分组

配置区域使用 Tab 分组展示，避免所有配置项堆在同一屏。

Tab 包括：

- `基础信息`
- `Wi-Fi 配置`
- `MQTT 配置`
- `ESP-NOW`

切换 Tab 时：

- 只显示当前分组内容。
- 底部操作按钮位置固定，不随内容高度跳动。
- 如果当前 Tab 内容超出区域高度，则在内容区域内出现滚动条。

### 基础信息

字段：

- `设备 ID`
- `设备角色`

设备角色选项：

- `主节点 / 网关`
- `从节点 / 采集节点`

### Wi-Fi 配置

字段：

- `Wi-Fi SSID`
- `Wi-Fi 密码`

说明：用于网关连接 2.4GHz Wi-Fi。

### MQTT 配置

字段：

- `MQTT Host`
- `MQTT 端口`
- `MQTT 用户名`
- `MQTT 密码`
- `Topic 前缀`

### ESP-NOW 配置

字段：

- `信道模式`
- `固定信道`

信道模式选项：

- `自动跟随 Wi-Fi`
- `固定信道`

固定信道范围：`1-13`

### 底部操作按钮

按钮固定在配置卡片底部。

按钮包括：

- `恢复`
- `预览并确认`

`恢复` 功能：

- 将表单恢复为当前设备已读取的配置。

`预览并确认` 功能：

- 读取当前表单生成完整 `config` 对象。
- 打开配置预览弹框。
- 弹框中以 JSON 格式展示完整配置。
- 确认后通过 `set_config` 写入设备 NVS。

## 右侧信息区域

### 硬件信息

标题：`硬件信息`

说明：`由 hello 命令返回。`

展示字段：

- 串口
- Type
- Chip
- USB
- MAC
- Flash
- 固件
- 状态

未选择设备时显示：`请选择设备`

## 底部状态栏

左侧显示当前操作状态，例如：

- `就绪`
- `扫描完成，发现 2 台设备。`
- `已选择: COM5`
- `COM5: 已读取当前配置。`
- `表单已恢复为当前配置。`
- `配置已写入，设备将重启。`

底部状态栏只显示当前操作状态，不再显示固定说明文案。

## 确认写入弹框

标题：`确认写入配置`

说明：`设备会将配置保存到 NVS，并重启生效。`

弹框内容：

- 以 JSON 格式展示将写入设备的完整 `config` 对象。

按钮：

- `取消`
- `写入并重启`

关闭方式：

- 点击取消。
- 点击遮罩。
- 按 `Esc`。

## 关于软件弹框

入口：左上角 `IOT Studio` 右侧三个点按钮。

弹框左侧显示：

- `IOT Studio`
- `ESP32-C3 配置工具`
- 当前软件版本：`v0.1.0`
- `UI Prototype`

弹框右侧显示：

- 应用名称：`IOT Studio`
- 软件版本：`v0.1.0`
- 构建类型：`Desktop UI Prototype`
- 目标平台：`Windows / macOS / Linux`
- 通信方式：`USB Serial / JSON Lines`
- 配置存储：`ESP32 NVS`

底部说明：

- `后续接入 Wails 后，可在这里展示真实版本号、构建时间和 Git Commit。`

关闭方式：

- 点击关闭按钮。
- 点击遮罩。
- 按 `Esc`。

## 关键交互规则

- 未选择设备时，配置表单不可编辑。
- 未选择设备时，恢复和确认按钮不可点击。
- 扫描后默认选中第一台设备。
- 点击设备列表中的设备后，表单和硬件信息同步刷新。
- 配置写入前必须经过确认弹框。
- 配置预览弹框展示完整 JSON，因此密码字段会按表单内容原样展示。
- 当前 UI 是静态原型，后续可迁移到 Wails 前端并接入真实串口 API。
